# SQL Advent Calendar

This repository contains my solutions for the [SQL Advent Calendar Challenge 2025](https://www.interviewmaster.ai/advent/) by Interview Master, created by Dawn Choo and Jeremiah Clark.

The challenge features 24 Christmas-themed SQL problems, released daily from December 1st to December 24th.

**Note:** All solutions are written in SQLite.
